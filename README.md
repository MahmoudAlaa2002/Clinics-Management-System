# Clinics Management System

A professional dental clinic management system designed to manage appointments, medical records, invoices, and analytics for dental clinics.

## Features

- Multi-clinic management
- Appointment scheduling
- Patient medical records
- Treatment plans
- Invoice and payment management
- Financial analytics dashboard

## Technologies Used

- Laravel
- MySQL
- Bootstrap
- jQuery
- Chart.js
